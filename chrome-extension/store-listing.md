Plain Tools – Offline PDF & Utilities

## Short Description (<=132 chars)
Launch 29+ local PDF tools, file converters, and diagnostics in one click. No uploads. No tracking.

## Full Description
Plain Tools gives you fast access to practical, privacy-first browser utilities.

Use the extension popup or dashboard to open:
- PDF workflows (merge, split, compress, convert, sign, OCR)
- File utilities (Base64, hashes, ZIP, image optimisation)
- Network diagnostics (site status, DNS lookup, IP tools)

Everything is designed around client-side processing where supported:
- no mandatory file upload step
- no analytics in the extension
- no background tracking

Key features:
- One-click launcher for plain.tools
- New tab dashboard with quick links
- Context menu shortcuts: Open Plain Tools, Search in Plain Tools
- Lightweight Manifest V3 build with minimal permissions

Open source:
https://github.com/Drgblack/plain-tools

Privacy policy:
https://plain.tools/privacy
