## Step 1 - Confirm portal requirements first

Before touching the file, collect the portal rules:

- maximum file size
- accepted file format
- per-file or total upload limits
- required page order
- naming rules

Many submission failures come from missing these basics.

## Step 2 - Build the document package

### When one final PDF is required

1. Merge source files in final reading order.
2. Check page sequence manually.
3. Compress once on the merged file.
4. Validate readability on key pages.

### When multiple PDFs are required

1. Split by page ranges based on the portal sections.
2. Name files with clear prefixes.
3. Keep each file under the portal limit.

If needed, use [How to split a PDF by pages](/learn/how-to-split-a-pdf-by-pages) for range handling.

## Step 3 - Validate quality before upload

Do a quick visual check:

- first page and last page
- any signature or stamp areas
- scanned pages with small text
- orientation and rotation consistency

If quality drops too far after compression, reduce compression level and re-check.

## Step 4 - Run privacy checks

For sensitive IDs, financial records, or supporting letters:

- minimise included pages
- remove unnecessary metadata
- avoid sharing full source bundles when only extracts are required

Use [How to remove PDF metadata](/learn/how-to-remove-pdf-metadata) for a metadata pass.

## Step 5 - Verify local processing claim

If privacy is critical, verify the tool path technically:

1. Open DevTools network tab.
2. Process a sample file.
3. Confirm there is no request carrying file bytes.

You can follow [/verify-claims](/verify-claims) to run this quickly.

## Step 6 - Final upload checklist

- Correct file names
- Correct file size limits
- Correct page order
- Correct language/translation attachments where required
- Local archive copy saved with date stamp

A short checklist saves more time than repeated portal retries.

## Practical note

Portal behaviour changes over time. Re-check official requirements each time you submit, even if you used the same portal last month.
